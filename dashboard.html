ℹ️ À Propos d'AgriDetect

AgriDetect est une plateforme d'intelligence artificielle dédiée à la détection et au diagnostic des maladies des cultures. Notre système utilise des modèles de deep learning avancés pour identifier avec précision les pathologies végétales.

Fonctionnalités principales :

  • ✅ Détection automatique des maladies par analyse d'images
  • ✅ Recommandations de traitements personnalisées
  • ✅ Chatbot intelligent pour assistance en temps réel
  • ✅ Base de données complète des maladies agricoles
  • ✅ Conseils de prévention et bonnes pratiques

Technologies utilisées :

  • 🔹 Backend : FastAPI (Python)
  • 🔹 Base de données : PostgreSQL
  • 🔹 Cache : Redis
  • 🔹 IA : TensorFlow/PyTorch
  • 🔹 Conteneurisation : Docker
